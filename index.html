Introduction

Tags

Tags create the logic and control flow for templates. they are denoted by curly braces and percent signs: {% and %}

The markup used in the tags does not produce any visible text. This means that you can assign variables and create conditions and loops without showing any of the liquid logic on the page.

Input

{% if user %}
 Hello {{ user.name }}
{% endif %}

Input

Hello Adam!

Tags can be categorize into three types:

  • Control flow
  • Iteration
  • Variable assignments

You can read more about each type of tags in their respective sections.

Filters

Filters change the output of a Liquid object. They are used within an output and are separated bay a |

Input 

{{ "/my/fancy/url | append: ".html" }}

Input

/my/fancy/url.html

Multiple filters can be used on one output.They are applied from left to right.

Input 

{{ "adama!" | capitalize | prepend: "Hello " }}

Input

Hello Adama!
